=================================================================
Install
=================================================================
1.Download plugins.
2.Set these files in "plugins" folder.
3.Start up server. RAI will install RedApple scripts automatically.
*Server will be reloaded. Don't worry, it's correct behavior.

1.プラグインをダウンロードする
2."plugins"フォルダーの中にダウンロードしたものを入れる
3.サーバーを起動する。RAIが自動的にRedAppleスクリプトをインストールします。
*サーバーがリロードされますが正常な動作です

=================================================================
Update
=================================================================
1.Download plugins.
2.Replace old RAI file and RAI.jar with new them.
3.Start up server and "update" command.
*Server will be reloaded. Don't worry, it's correct behavior.

1.インストール時同様にファイルをダウンロードする
2.古いRAIフォルダ・RAI.jarを新しいものと入れ替える
3.サーバーを起動し、コンソール上で"update"コマンドを実行する
*サーバーがリロードされますが正常な動作です

=================================================================
Play
=================================================================
1.Put players on Golden Block.
2."/prepare"
3."/start"

1.参加者を金ブロックの上に乗せる
2."/prepare"コマンドで参加者を確定する
3."/start"コマンドでゲームを開始する
